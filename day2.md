# Day 2 - Prompt Engineering

## Lazy Prompt
**Prompt:**
Create an image explaining Prompt Engineering.

**Explanation:**
This prompt is short and general. It gives very little information, so the AI creates a basic output.

## Engineered Prompt
**Prompt:**
Act as an expert AI educator and infographic designer. Create a colorful infographic explaining Prompt Engineering for beginners. Include what Prompt Engineering is, why it is important, five tips, examples, and a modern design.

**Explanation:**
This prompt is detailed and structured. It gives the AI clear instructions, so the output is more informative and visually appealing.

## Comparison
The engineered prompt produced a better result because it included detailed instructions, while the lazy prompt produced a basic output.

## What I Learned
- Prompt engineering improves AI responses.
- Detailed prompts create better results.
- Clear instructions help AI understand the task.
